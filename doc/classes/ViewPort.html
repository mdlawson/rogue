    Class: ViewPort

    Defined in: src/rogue.coffee

    Overview

    Viewport class, most games will use this. Expands the game world to be larger than the canvas.

    Instance Method Summary

    • - (void) constructor(options) Constructor
    • - (void) add(entity) Add an entity or an array of entities to the viewport, will be updated and rendered with the viewport, and position is drawn relative to the viewports position
    • - (void) update(dt) Updates all entities within the viewport.
    • - (void) draw() Draws all entities within the viewport.
    • - (void) move(x, y) moves the view area by [x,y]
    • - (void) moveTo(x, y) moves the view area to [x,y]
    • - (void) follow(entity) Makes the view follow an entity, so that entity is always in the center
    • - (void) forceInside(entity, buffer = 0) prevents an entity from going outside of the view area.
    • - (Rect) rect() used for checking the visibility of objects
    • - (Bool) visible(entity) Checks if an entity is currently visible.
    • - (void) keepInBounds()
    • - (void) find(components, ex)
    • - (void) close(entity)

    Constructor Details

    - (void) constructor(options)

    Parameters:

    • options (Object) — options for setting up the viewport

    Options Hash: (options):

    • parent (Game) — the parent game instance to attach the viewport to
    • canvas (Canvas) — the canvas this viewport should render on, defaults to the parents canvas
    • width (Int) — the width of the viewport, defaults to the canvas width
    • height (Int) — the height of the viewport, defaults to the canvas height
    • viewWidth (Int) — the width of the viewable area, defaults to width
    • viewHeight (Int) — the height of the viewable area, defaults to height

    - (void) update(dt)

    Updates all entities within the viewport. The viewport update function should be called from your states update function

    Parameters:

    • dt (Float) — dt, the time elapsed between ticks. all update functions should be passed dt if physics is needed.

    - (void) draw()

    Draws all entities within the viewport. The viewport draw function should be called from within your states draw function.
